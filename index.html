Git commands

Create directory: git init
Configure git my name: git config --global user.name 'Youri van Deutekom'
Configure git my email: git config --global user.email 'yourivandeutekom@gmail.com'
add file to commit(track): git add filename.extension
Ask for status: git status
Remove file to commit(track): git rm --cached filename.extension
Add all from extension to commit(track): git add *.html
Add all to commit(track): git add .
commit: git commit -m 'comment'
clear console: clear

Ignore files to be commited.

Create ignore file: touch .gitignore
Add the file to be ignored on each line, for a folder enter /foldername
