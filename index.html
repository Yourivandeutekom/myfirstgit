Git commands

Create directory: git init
Configure git my name: git config --global user.name 'My name'
Configure git my email: git config --global user.email 'My email'
add file to commit(track): git add filename.extension
Ask for status: git status
Remove file to commit(track): git rm --cached filename.extension
Add all from extension to commit(track): git add *.html
Add all to commit(track): git add .
commit: git commit -m 'comment'
clear console: clear

Ignore files to be commited.

Create ignore file: touch .gitignore
Add the file to be ignored on each line, for a folder enter /foldername

Add branches.

Add branches: git branch NAMEOFBRANCH
Switch to branches: git checkout NAMEOFBRANCH
Merge branch to the master step 1: git checkout master
step 2: merge branch 'NAMEOFBRANCH'
step 3: under the line add a commit message.
step 4: press on 'esc' and ":wq"

Add remote repository.

Create new repository github
Follow the instructions on the website! from a existing git init follow the steps from: git remote add origin LINK
then push the git: git push -u origin master
